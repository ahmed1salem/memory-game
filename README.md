# لعبة الذاكرة 🧠

لعبة مبنية بـ HTML, CSS, JavaScript حيث يجب عليك إيجاد الأزواج المتطابقة من البطاقات!

## كيفية اللعب 🕹
1. انقر على بطاقة لقلبها.
2. ابحث عن الزوج المطابق.
3. فاز عندما تجد جميع الأزواج بأقل عدد من المحاولات والوقت.
